# Screenshots

Place screenshots in this directory.

Recommended files:

- `overview.jpg` — combined screenshot showing Taskshell UI and MCP tool calls.
- `taskshell-service.jpg` — Taskshell service status screen.
- `taskshell-local-test.jpg` — local test output screen.
- `mcp-client-tools.jpg` — MCP client tool call screen.

For GitHub README display, keep images compressed and preferably below 1 MB each.
